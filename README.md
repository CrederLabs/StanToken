# STAN Token Contracts
STAN Token Contracts for RWA Ecosystem

The STAN token is an EIP20 token.

The total supply of the token is 1,000,000,000, and each token is divisible up to 18 decimal places.

## Details
- Decimals: 18
- Name: Station Token
- Symbol: STAN
